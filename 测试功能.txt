4.4 版本更新 (cmd=604)
主控服务器通知控制板进行版本更新 ，下发新版本号及下载地址。
请求参数
参数名	数据类型	必填	取值/说明
cmd	int	是	固定值   604
token	string	是	平台生成的唯一鉴权令牌
mainVer	int	是	新版本的主版本号
downUrl	string	是	新版本固件的下载链接
请求⽰例
{
"cmd" : 604,
"token" : "xyz123",
"mainVer" : 3,
"downUrl": "http://192.168.1.100/firmware/v3.0.bin" }
成功响应
{
"code" : 0,
"cmd" : 602,
"msg" : "success" }
